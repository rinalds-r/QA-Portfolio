# Test Plan

## Project Information

**Project Name:** Shopify Sauce Demo Store

**Application Under Test:**
https://sauce-demo.myshopify.com/

**Application Type:**
Demo E-commerce Website

**Testing Type:**
Manual Functional Testing

---

# Objective

The objective of this project is to verify that the main functionality of the Shopify demo store works correctly and provides a smooth user experience.

---

# Scope

The following functionality will be tested:

- Home page
- Navigation menu
- Product catalog
- Product pages
- Shopping cart
- Search
- Customer login
- Password recovery
- Contact forms

---

# Out of Scope

The following areas are not covered:

- Payment gateway
- Performance testing
- Load testing
- Security testing
- Mobile application
- Admin panel
- Database testing
- Source code review

---

# Test Environment

Operating System:
Windows 10

Browser:
Google Chrome (latest stable version)

Network:
Stable Internet connection

---

# Test Types

The following testing types will be performed:

- Functional Testing
- UI Testing

---

# Entry Criteria

Testing starts when:

- Website is available
- Browser is installed
- Test environment is ready

---

# Exit Criteria

Testing is completed when:

- All planned test cases are executed
- Critical bugs are documented
- Test execution report is completed

---

# Risks

Possible risks:

- Website content may change during testing
- Temporary server downtime
- Browser updates affecting UI
- Demo store limitations

---

# Deliverables

The project will include:

- Test Plan
- Checklists
- Test Cases
- Test Execution Reports
- Bug Reports
